# Python Assignment - Basic Programs

## Description
This project contains two simple Python programs:

1. A basic calculator that performs addition, subtraction, multiplication, and division.
2. A program that takes first name and last name as input and prints a welcome message.

These programs are created as beginner-level Python practice tasks.## Files Included

- task1.py  → Calculator Program
- task2.py  → Full Name Greeting Program

---

## Task 1 - Calculator Program

This program:
- Takes two numbers as input
- Performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays all results

### How to Run

Open terminal in the project folder and run:

python task1.py
### Example

Input:
Enter the first number: 10
Enter the second number: 5
Output:
Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2
---

## Task 2 - Greeting Program

This program:
- Takes first name and last name as input
- Combines them into full name
- Prints a welcome message

### How to Run

python task2.py
### Example

Input:
Enter your first name: Mantra
Enter your last name: Patel
Output:
Hello, Mantra Patel! Welcome to python program.
